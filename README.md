# Clinical Data Management System 🏥💻

This is a professional desktop application designed for clinical environments to manage patient records efficiently. The project implements the **Model-View-Controller (MVC)** architectural pattern to ensure a clean separation between data logic and user interface.

## 🚀 Features
- **Patient Registration:** Comprehensive form to add name, ID, service, and age.
- **Search System:** Fast retrieval of patient data by ID.
- **Data Integrity:** Validation systems to prevent duplicate records.
- **Delete Functionality:** Capability to remove records from the system.
- **User-Friendly Interface:** Developed with Qt Designer for an intuitive experience.

## 🛠 Tech Stack
- **Language:** Python 3.12
- **GUI Framework:** PyQt5
- **Design Tool:** Qt Designer (.ui files)
- **Architecture:** MVC Pattern
